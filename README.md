Overview
Multiple Linear Regression – Multi-Channel Marketing Analysis
Project Overview
In this project, you will analyze a marketing dataset using Python and statsmodels to build a Multiple Linear Regression model. You will learn how to check for multicollinearity, implement multivariate OLS regression, validate advanced model assumptions, and interpret adjusted statistics to drive data-informed budget decisions. This project strengthens your ability to model real-world scenarios where multiple factors influence outcomes simultaneously.

Step-by-Step Project Tasks
• Load the dataset and perform initial exploratory data analysis
• Check for multicollinearity among independent variables (TV, Radio, Social Media) using correlation matrices and VIF
• Build a Multiple Linear Regression model using statsmodels to predict Sales
• Evaluate model performance using Adjusted R-squared and individual predictor p-values
• Create diagnostic plots to validate Linearity, Normality, and Homoscedasticity of residuals
• Interpret coefficients in context: explain the effect of a one-unit increase in one channel's spend while holding others constant
• Synthesize findings into a clear, business-ready recommendation

Project Goal
By the end of this project, your analysis should:
• Successfully detect and address multicollinearity issues among predictors
• Fit a statistically robust Multiple Linear Regression model
• Justify variable inclusion/exclusion based on p-values and Adjusted R-squared
• Verify regression assumptions using residual diagnostic visualizations
• Clearly explain coefficient interpretations to stakeholders (e.g., "Holding Radio and Social Media spend constant, each additional $1K spent on TV is associated with a $X increase in Sales")
• Deliver a prioritized, evidence-based recommendation for marketing budget allocation
